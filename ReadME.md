# ⠀⠀⠀ 	

# &#x20;   					     Описание проекта

Проект представляет собой систему удалённого мониторинга объектов. В реальности такая система могла бы использоваться диспетчерскими службами для контроля за работой транспорта, доставки и складов. В рамках практики мы эмулируем работу датчиков, установленных на 4 типах объектов. Каждый объект отправляет телеметрию, которая обрабатывается, сохраняется и отображается на веб-карте. Система автоматически детектирует аварийные ситуации и отправляет предупреждения в Telegram.

# &#x09;			        	Установка





Скачать репозиторий с github
Распаковать в нужную папку на пк

Перейти в папку backend в терминале(cmd)
скачать необходимые библиотеки
pip install -r requirements.txt

Затем перейти в папку frontend в терминале
Установите зависимости Node.js с помощью команды npm install

Проект установлен

# &#x09;				 	  Запуск

перед тем как запустить проект включи VPN для работы Telegram-бота. Затем пропишите /start - для получения уведомлений

запустить run.bat
Проект запущен.
